# Troubleshooting

## App Won't Start

1. Close the app
2. Uninstall and reinstall using Setup.exe
3. Restart Windows

## No Clips Appearing

1. Copy new text to test
2. Check Settings → Polling is enabled
3. Restart the app

## App Freezes

1. Close and restart
2. Check: Do you have free disk space (500MB+)?
3. Restart Windows

## Need More Help?

Check QUICK_START.md or DEPLOYMENT.md for setup details.
