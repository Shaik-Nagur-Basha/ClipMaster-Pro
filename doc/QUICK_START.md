# Quick Start Guide

## What is ClipMaster Pro?

Clipboard manager that captures everything you copy.

## Install in 30 Seconds

1. Run `ClipMaster-Pro-Setup.exe`
2. Click Install
3. Done!

## Start Using in 1 Minute

1. Copy some text (Ctrl+C)
2. Open ClipMaster Pro
3. Your text appears automatically
4. Search, tag, or star as needed

## What You Can Do

- **Capture** — Every copy saved automatically
- **Search** — Find clips instantly
- **Tag** — Organize by category
- **Favorite** — Mark important clips with ⭐
- **Delete** — Soft delete with restore
- **Sync** — Backup to MongoDB (optional)

## Common Tasks

### Save Important Clip
Click ⭐ star icon

### Find a Clip
Type in search box

### Organize with Tags
1. Click tag icon
2. Select or create tag
3. View clips by tag

### Restore Deleted Clip
1. Go to Recycle Bin
2. Click restore icon

### Edit a Clip
1. Click pencil icon
2. Modify text
3. Press Ctrl+Enter to save

## Pro Tips

- Default polling: 600ms (good for most)
- Increase max entries for larger history
- Use tags by project/category
- Enable auto-launch at startup
